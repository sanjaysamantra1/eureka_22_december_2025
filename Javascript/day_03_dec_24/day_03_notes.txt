undefined vs null
==================
-when a variable is declared without any value, javascript assigns 'undefined' value to that variable.
-'null' is an intensional value assigned by developers
-typeof 'undefined' is undefined, typeof 'null' is object

Q. typeof 'null' is object, is 'null' a non-primitive value?
Note: even though typeof 'null' is object, null is a primitive value
